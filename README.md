# JSocials

A platform for connecting people and creating relationships.

## Setup

Set all the environment variables in `.env`, then run

```sh
docker-compose up
```

## Deployment

Export all the necessary env variables  
The app's port is set by the `PORT` variable

```sh
./build.sh
cd server
yarn start
```
